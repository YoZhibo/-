# 嵌入式定位  

## 一、开发环境  
软件：Arduino  
硬件：GT-U7 GPS模块，ESP8266 WiFi模块，面包板，杜邦线  
编程语言：C  

## 二、主要功能  
1、通过GPS模块串口获取GPS信息，输出到串口监视器  
2.WiFi模块连接wifi，实现上网。  
3.启动本地服务器和客户端，传输GPS信息。
